> هدف:
تولد پروژه در Local و معرفی آن به Remote

گام 1 — ساخت پروژه

`mkdir my-project`
`cd my-project`

___

گام 2 — مقداردهی Git

[[Git init]]

___

گام 3 — اولین فایل

`touch README.md`

___

گام 4 — اولین [[git commit]] (خیلی مهم)

`[[Git add]] .`
`git commit -m "Initial commit"`

___

گام 5 — اتصال به [[Remote]]

`git remote add [[origin]] git@github.com:user/repo.git`

___

گام 6 — [[Git push]] اولیه

```
[[git branch]] -M main
[[Git push]] -u origin main
```

___
