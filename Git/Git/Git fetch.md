تفاوت Fetch و [[Git pull]]

* Git Fetch: فقط لیست تغییرات را از سرور می‌گیرد (بدون دست زدن به کدهای شما). کاملاً امن است.
* Git Pull: تغییرات را می‌گیرد و بلافاصله با کد شما ادغام ([[Git merge]]) می‌کند
___

**دستور git fetch (باخبر شدن)**


​این دستور کمی زیرکانه است. وقتی fetch می‌کنی، گیت فقط می‌رود سرور را چک می‌کند تا ببیند آیا بقیه (همکارانت) کدی فرستاده‌اند یا نه. اگر کد جدیدی باشد، آن‌ها را دانلود می‌کند و در یک گوشه نگه می‌دارد، اما آن‌ها را با کد فعلی تو ترکیب نمی‌کند.
